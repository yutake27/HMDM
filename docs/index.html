HMDM

Homology models dataset for model quality assessment

Abstract

Homology models dataset for model quality assessment (HMDM) is a benchmark dataset for model quality assessment that contains highly accurate model structures using homology modeling.

There are two version of datasets: HMDM-single and HMDM-multi. Each dataset contains 100 targets, and 50 to 150 model structures are included in one target.

Download

Following datasets are currently available. Note that the size of the dataset is large. The list of datasets can be seen here.

  • HMDM-single (2021-12-08) (only single-domain targets)
  • HMDM-multi (2021-12-08) (only multi-domain targets)
